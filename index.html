Methodology & Data Notes
Search terms
"[City] World Cup 2026" — identical phrasing across all 16 cities for direct comparability.
Scoring
Google Trends returns relative interest 0–100. All 16 cities are normalised across batches using an anchor term so scores are truly comparable.
Weekly score
7-day rolling window. Reflects current tournament momentum and recent match activity.
Cumulative total
Average of weekly snapshots since tournament start (11 Jun 2026). Stored and averaged each Saturday by the GitHub Action.
Data freshness
Scores are fetched automatically every Saturday at 08:00 UTC via GitHub Actions and written to data/data.json.
